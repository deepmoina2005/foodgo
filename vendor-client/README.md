# FoodGo Vendor Client

Vendor dashboard app for restaurant profile management, menu CRUD, incoming orders, reports, reviews, payouts, and notifications.

## Setup

```bash
cd vendor-client
npm install
npm run dev -- --port 5174
```

## Features

- Vendor login and signup
- Vendor dashboard, profile, restaurant setup, documents upload
- Food CRUD, stock toggle, active/inactive status
- Accept/reject orders and update statuses
- Reports, reviews, payouts, settings, notifications

## API

- `VITE_API_BASE_URL=http://127.0.0.1:8000/api`
