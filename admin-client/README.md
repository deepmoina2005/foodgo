# FoodGo Admin Client

Admin dashboard app for managing customers, vendors, restaurants, delivery partners, categories, foods, banners, orders, coupons, payouts, complaints, reports, and notifications.

## Setup

```bash
cd admin-client
npm install
npm run dev -- --port 5175
```

## Features

- Admin login only
- Dashboard stats and charts
- Customer block/unblock
- Vendor and delivery approval workflows
- Restaurant moderation
- Category, food, banner, coupon, payment, payout, complaint, and report management

## API

- `VITE_API_BASE_URL=http://127.0.0.1:8000/api`
