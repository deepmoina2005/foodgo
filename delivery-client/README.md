# FoodGo Delivery Client

Delivery partner dashboard app for assigned orders, KYC upload, delivery history, earnings, ratings, and notifications.

## Setup

```bash
cd delivery-client
npm install
npm run dev -- --port 5176
```

## Features

- Delivery partner login and signup
- Profile and KYC upload
- Assigned orders and delivery status updates
- Delivery history, earnings, ratings, notifications, settings

## API

- `VITE_API_BASE_URL=http://127.0.0.1:8000/api`
