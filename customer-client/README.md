# FoodGo Customer Client

Customer dashboard app for browsing restaurants and foods, managing cart and checkout, tracking orders, reviews, complaints, and notifications.

## Setup

```bash
cd customer-client
npm install
npm run dev -- --port 5173
```

## Tech Stack

- React
- Vite
- Redux Toolkit
- React Router DOM
- Axios
- Tailwind CSS
- Lucide React
- React Hook Form
- Toast notifications

## Features

- Customer login and signup
- Dashboard, profile, addresses, cart, checkout, orders, tracking, invoice, reviews, complaints, notifications
- Restaurants and foods browsing
- COD-only checkout
- Database notifications

## API

- `VITE_API_BASE_URL=http://127.0.0.1:8000/api`
